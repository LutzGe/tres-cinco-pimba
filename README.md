# Três Cinco Pimba (FizzBuzz)

### Estrutura:
```
app
├── composer.json
├── Dockerfile
├── phpunit.xml
├── src
│   └── CountPimba.php
└── tests
    └── CountPimbaTest.php
```

### Usando o Docker:

`docker-compose up -d`

`docker exec -it trescincopimba bash`

<b> Testes Unitários </b>
Dentro da pasta app/

`composer run test`


<b> Script main.php </b>
Dentro da pasta app/src/

`php main.php`

### Usando o PHP e Composer local
<b> Testes </b>
Dentro da pasta app/

`composer run test`

<b> Script main.php </b>
Dentro da pasta app/src/

`php main.php`
